# ApexTip AI Backend

**ApexTip AI** es una plataforma premium y privada orientada a las predicciones deportivas y análisis predictivo asistido por Inteligencia Artificial.

Este repositorio contiene el backend propietario del proyecto, encargado de orquestar la infraestructura, los modelos analíticos de IA y la gestión de la comunidad de tipsters.

## Descripción General
El sistema procesa datos estadísticos deportivos en crudo y utiliza motores de IA generativa con outputs estructurados para calcular pronósticos y probabilidades matemáticas. Además, proporciona la infraestructura backend para una red social donde los tipsters profesionales pueden compartir sus análisis, y los usuarios premium pueden consumirlos.

## Stack Tecnológico Principal
- **Core:** FastAPI (Python 3.11+) y Arquitectura Limpia.
- **Base de Datos:** PostgreSQL (vía Supabase).
- **Seguridad:** Autenticación de Firebase Auth.
- **Inteligencia Artificial:** Google Gemini SDK & Model Context Protocol (MCP).
